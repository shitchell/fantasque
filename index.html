how to use

  1. add the following code to your <head> 
    <link rel="stylesheet" href="https://shitchell.github.io/fantasque/Webfonts/FantasqueSansMono-Regular-decl.css" />
<link rel="stylesheet" href="https://shitchell.github.io/fantasque/Webfonts/FantasqueSansMono-Bold-decl.css" />
<link rel="stylesheet" href="https://shitchell.github.io/fantasque/Webfonts/FantasqueSansMono-Italic-decl.css" />
<link rel="stylesheet" href="https://shitchell.github.io/fantasque/Webfonts/FantasqueSansMono-BoldItalic-decl.css" />
  2. use the following rule in your css 
    font-family: "Fantasque Sans Mono", monospace;
